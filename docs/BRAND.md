# TrimDoctor Brand Guidelines

## Identity
- **Name:** TrimDoctor
- **Domain:** trimdoctor.com
- **Tagline:** "Doctor-guided weight loss, delivered."
- **Founder:** Michael Gonzalez (@teaceo, 491K followers, FitTea $100M+ exit)

## Logo
- **Mark:** Forest green rounded square with "T" letterform + gold medical cross
- **Variants:** Color (on light), Reversed (on dark), Mark-only (favicon/app icon)
- **Clear space:** Minimum 0.5x mark width on all sides
- **SVG source:** See `src/components/Logo.tsx`

## Typography
| Role | Font | Weights | Usage |
|------|------|---------|-------|
| Display | Playfair Display | 400, 700, 800 | Hero headlines, section titles, pitch deck |
| Body | Figtree | 300-800 | Everything else — body, labels, buttons, nav |
| Data | JetBrains Mono | 400, 500, 600 | Prices, stats, IDs, code, metrics |

## Design Principles
1. **Clinical warmth** — Medical credibility meets approachable DTC brand
2. **Grain texture** — SVG noise overlay on all pages for premium depth
3. **Glass morphism** — Nav bar with blur-on-scroll effect
4. **Premium cards** — 20px border-radius, 1px borders, hover lift shadows
5. **Gold as accent** — Stars, step numbers, premium badges, shimmer text
6. **Dark theme for admin** — All internal/operational tools use dark backgrounds

## Tone of Voice
- **Patient-facing:** Warm, reassuring, clinical but not intimidating
- **Ad copy:** Conversational, authentic — never salesy
- **Admin/internal:** Professional, concise, data-driven
- **Legal:** Template-quality, attorney-reviewed
